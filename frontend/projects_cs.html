Computer Science Projects

Implementation and Performance Analysis of Asymmetric Cryptosystems

Role: Creator, Project Lead

C++, Java

An implementation and runtime comparison of the cryptosystems RSA and ElGamal, investigating whether RSA is favored in industry due to greater efficiency. Implementation is provided in both Java and C++, using one non-compiled and one compiled language to produce and compare runtime data for each cryptosystem.

Finance Tracker Application

Role: Creator

C++, Qt Creator

An application for MacOS devices for keeping track of income and expenses in an organized environment with graphing capabilities to help visualize the rate of finance flow. Written in C++ and using Qt Creator GUI. Features include user credential management and encryption, customizable categories to organize data, and financial graphing functionality.

Test Automation and Documentation for Pineapple

Role: Test Automation Programmer, Documenter

Python, Bash

Documentation with guided, step-by-step instructions detailing how to run and plot automated experiments in python through the terminal to test and compare the throughput and latency of Pineapple, against other distributed systems. More information on Pineapple can be found in the paper "Pineapple: Unifying Multi-Paxos and Atomic Shared Registers".

Quicksort Partition Implementation and Runtime Analysis

Role: Creator

Java

An implementation and comparison of two different partition methods for the Quicksort algorithm: separation and swap. Code implementation is in Java and is based on the pseudocode in Jeff Erickson's Algorithms.
